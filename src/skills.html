Навыки

JavaScript

Могу создавать интерактивные веб-приложения, обрабатывать события, манипулировать DOM и работать с асинхронными запросами. Также знаком с основными концепциями функционального и объектно-ориентированного программирования в JavaScript.

html

Я уверенно создаю разметку для веб-страниц, используя основные теги и атрибуты. Могу правильно структурировать контент, включая текст, изображения, ссылки и формы. Также знаком с семантической разметкой и основными принципами доступности в HTML

git

Я умею создавать репозитории, клонировать проекты, добавлять, коммитить и пушить изменения. Знаком с базовыми командами, такими как git add, git commit, git push, git pull, git clone. Осознаю основные концепции ветвления и слияния, а также могу создавать и переключаться между ветками.

gulp

Я умею создавать и настраивать задачи для автоматизации сборки проектов, такие как компиляция препроцессоров CSS (например, Sass), минификация и конкатенация файлов, оптимизация изображений, запуск тестов и другие действия. Также знаком с использованием плагинов Gulp и умею интегрировать их в рабочий процесс.

TypeScript

Я могу создавать базовые приложения на TypeScript, определять типы данных для переменных, параметров функций и возвращаемых значений, а также использовать интерфейсы для описания структуры объектов. Однако мой опыт ограничен использованием базовых функций и концепций, и мне нужно продолжать изучение для более глубокого понимания продвинутых возможностей TypeScript, таких как дженерики, декораторы и продвинутая работа с типами.

redux

Я понимаю основные концепции управления состоянием в Redux, такие как хранилище (store), действия (actions), редюсеры (reducers) и подключение компонентов к хранилищу через connect. Я могу создавать базовые приложения с использованием Redux, реализовывать базовые операции чтения и записи в хранилище, а также следить за изменениями состояния приложения. Однако мой опыт ограничен более сложными аспектами, такими как использование middleware, нормализация состояния и оптимизация производительности.
